# How to Run the Encryption CLI

Follow these steps to download and run the Rust encryption project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/Daw155/encrypt.git
   ```

2. Navigate into the project directory:
   ```bash
   cd encrypt
   ```

3. Run the project using Cargo:
   ```bash
   cargo run
   ```

4. Follow the interactive prompts in the terminal to choose a cipher, provide text (or a file), and supply a key to perform the encryption or decryption.
